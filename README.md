# Next Topics

Et pedagogisk prosjekt bygget med **Next.js 16** og **Tailwind CSS v4**, laget for å presentere **dagens topic** — korte, konkrete og forståelige temaer innen moderne webutvikling.  
Hver topic skal ta maks **20 minutter**, og fungerer både som *dagens recap* og som *mini-forelesning*.

---

## Teknologier & rammeverk

| Teknologi | Versjon | Formål |
|------------|----------|--------|
| [Next.js](https://nextjs.org/) | **v16** | App Router, rask SSR/SSG, innebygget metadata og routing |
| [Tailwind CSS](https://tailwindcss.com/) | **v4.x** | Stiloppsett i `globals.css`, fokus på enkelhet og tema-variabler |
| [Supabase](https://supabase.com/) | — | Database, autentisering og lagring |
| JavaScript| — | Prosjektet skal være lett å lese og forstå |
| Vercel + GitHub | — | Deploy & samarbeid |

---

## Formål

Dette prosjektet brukes til å:

1. **Gjøre læring praktisk.** Hver topic er kort, konkret og kan gjennomføres på 10–20 minutter.  
2. **Vise god praksis i Next.js og Tailwind.** Både nybegynnere og viderekomne får gjenbrukbare mønstre.  
3. **Bygge et levende arkiv.** Alle topics ligger som separate sider i App Router-strukturen.  
4. **Være lett å dele og bidra til.** Én README, ryddig kode, og enkle prinsipper.
